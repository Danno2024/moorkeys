# MoorKeys

MoorKeys is a Laravel-based SaaS platform for creating, managing, and validating software activation keys.

Designed for developers and businesses who need a simple licensing and activation system for:

- Desktop applications
- SaaS applications
- Mobile apps
- WordPress plugins
- Laravel packages
- Custom software solutions

## Features

✅ User registration and authentication  
✅ Create software products  
✅ Generate activation keys  
✅ Key activation API  
✅ Key expiry management  
✅ Licence status tracking  
✅ Customer management  
✅ Admin dashboard  

## Tech Stack

- Laravel
- MySQL
- Blade / Livewire
- REST API
- Tailwind CSS

## Installation

Clone the repository:

git clone https://github.com/username/moorkeys.git

Install dependencies:

composer install

Create environment:

cp .env.example .env

Generate key:

php artisan key:generate

Run migrations:

php artisan migrate
